Procjam 2019 - Quest Tree

by Nikolaj Baer

Your Quest

Hi There!

This is a 2019 ProcJam project where I am trying to dynamically generate a "Quest" for an RPG-like game, using a dependency tree as my model. I don't think this is particularily new territory, but here are my rough goals.

Core Goals

  • Create a dynamic quest with multiple sub-quests buit out of actions and pickups
  • Create a top-level narrative summary that makes sense

Stretch Goals

  • Create multiple resolutions to quests, so there are multiple ways to solve it
  • Create some sort of storyline that fits, and possibly intertwines with the quest
  • Make it fun

Hopefully I get some of this done!,
- Nikolaj

PS - The awesome sprites are from Kenney.nl, and game is built on the very elegant Crafty.js! Check out my source code if you are curious!
